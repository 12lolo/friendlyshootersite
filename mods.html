Installing Mods

  1. Download the mod archive and extract it.
  2. Read the mod's README for compatibility notes and required game version.
  3. Copy mod files into your FriendlyShooter game folder (backup saves first).
  4. Run the game and enable mods if required by the mod's instructions.

Submit a Mod

To submit a mod for listing here, create a ZIP containing the mod and a short README, then email the creator or open an issue on the repo with a link. Include compatibility notes and installation steps.
