Mod Launcher (MelonLoader) — Download & Installation

Note: The Mod Launcher (MelonLoader) is required to install and run community mods.

  1. Download the Mod Launcher (MelonLoader) from: melonwiki.xyz
  2. Open the installer and click "Add Game Manually" in the lower-right corner.
  3. Browse to and select FriendlyShooter.exe in your game folder.
  4. Select the FriendlyShooter entry that was added in the installer, then click Install.
  5. Move the mod file into the game's Mods folder. If the folder does not exist, create it or run the game once to generate it.

For Linux users (Steam Proton / Wine), add this to the game's launch options:

WINEDLLOVERRIDES="version=n,b"
